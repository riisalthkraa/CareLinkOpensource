## Technologies Utilisées

| Catégorie | Technologies |
|-----------|-------------|
| **Frontend** | React 18, TypeScript 5.3, Vite 5 |
| **Desktop** | Electron 28 |
| **Base de données** | SQLite (sql.js) |
| **IA/ML** | Python FastAPI, Sentence-BERT |
| **Sécurité** | bcrypt, AES-256, safeStorage Electron |
